Make a note

This space is for you to write down anything you remember about the event or events. These details won't be sent anywhere, so you can use this space to write down anything you might want to come back to.

For example, you may wish to include details of:

  • What exactly happened: what was said or done
  • The name of the individual responsible, if applicable
  • The name of the individual targeted, if not yourself
  • How many offence(s) occurred
  • Where and when the offence(s) took place
  • Names of any witnesses to the offence(s)
